This project focuses on validating and analyzing Indian PAN card numbers using SQL.
The objective is to identify valid, invalid, and duplicate PAN numbers using pattern matching and data validation techniques.
##Problem Statement
Organizations often store PAN numbers in large databases.
Incorrect formats, duplicates, and invalid entries can create compliance and reporting issues.
This project aims to:
Validate PAN format using SQL
Detect invalid PAN numbers
Identify duplicate entries
Improve overall data quality
